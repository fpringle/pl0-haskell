type PL0Parser a = Parsec [Token] () a Source #

A parser that represents a transformation from a list of Tokens.

If p is a parser of type PL0Parser (Block Identifier) and tks is a token list of type [Token], then parse p fp tks will give a value of type Either ParseError Block.
